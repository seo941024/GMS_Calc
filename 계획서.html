방학 및 향후 학습 계획서

01

학습 및 프로젝트 수행 현황

2026년 3월 입학 이후 현재까지 프로그래밍 기초 역량 강화와 실무형 프로젝트 수행을 병행하며 학업에 임하고 있다. 전공 이론 학습에 그치지 않고 직접 결과물을 만들어 보는 경험을 중시하여, 각 월별로 구체적인 프로젝트 목표를 설정하고 이를 달성해 나가는 방식으로 학습을 이어가고 있다.

입학 초기인 3월에는 Python 프로그래밍, 데이터베이스, 알고리즘 등 기초 전공 과목을 집중적으로 학습하였다. 입학 이전 프로그래밍 경험이 거의 없었던 만큼, 수업 복습과 연습문제 반복 풀이를 통해 기본기를 다지는 데 주력하였다. 낯선 개념이 등장할 때마다 공식 문서와 예제 코드를 적극 참고하며 스스로 문제를 해결하는 습관을 형성하였고, 이와 동시에 정보처리기사 필기시험 대비를 위한 전공 이론 학습도 체계적으로 진행하였다.

4월에는 HTML, CSS, JavaScript를 활용한 웹 개발로 학습 범위를 확장하였다. 단순히 따라 만드는 수준을 넘어, 개인 프로젝트로 To-Do List 웹 애플리케이션을 기획부터 구현까지 직접 완성하였다. 사용자 인터페이스 설계, 이벤트 처리, 로컬 데이터 관리 기능을 구현하는 과정에서 프론트엔드 개발의 전반적인 흐름을 체험할 수 있었다. 아울러 외부 공모전에 참가하여 팀 단위로 프로젝트를 기획하고 개발하는 협업 경험도 쌓았다.

5월에는 HTML과 JavaScript를 기반으로 한 웹 게임 개발 프로젝트에 참여하였다. 캐릭터 시스템, 전투 로직, 인게임 UI 등 다양한 기능을 팀원과 역할 분담하여 구현하였으며, 이 과정에서 코드 품질 관리와 협업 시 발생하는 충돌 해결 방법을 직접 경험할 수 있었다. 또한 정보처리기사 필기시험에 응시하였으며, 시험 직후부터 약 2주간 SQLD 자격증 취득을 목표로 데이터베이스 이론과 SQL 문법을 집중 학습하여 시험에 응시하였다.

최근에는 팀 프로젝트로 진행된 AI 기반 주차장 관리 시스템 개발을 성공적으로 마무리하였다. Python, Flet, SQLite, YOLO, EasyOCR을 기술 스택으로 활용하여 차량 번호판 실시간 인식, 입·출차 데이터 저장 및 조회, GUI 기반 관리자 화면 구현 등을 진행하였다. 단순 기능 구현에 그치지 않고 실제 운영 환경을 가정하여 예외 처리와 데이터 정합성까지 고려하며 개발에 임하였으며, 팀 내 핵심 개발 파트를 책임지고 주도하여 완성도 높은 결과물을 도출하였다.

방학 중 자격증 학습 계획

방학 기간 동안 자격증 취득을 통해 실력을 객관적으로 검증하고, 이를 포트폴리오와 연계하여 취업 경쟁력을 강화할 계획이다. 각 자격증의 시험 일정에 맞추어 월별로 집중 학습 기간을 설정하고, 기출문제 풀이와 이론 학습을 병행하는 방식으로 준비할 예정이다.

시기 자격증 학습 방향
2026년 7월 정보처리기사 실기 알고리즘·SQL·업무 프로세스 유형별 기출 반복 풀이, 약점 파트 집중 보완
2026년 8월 ADsP (데이터분석 준전문가) 데이터 분석 기획·처리·탐색 이론 학습 및 통계 개념 정리, 실전 문제 풀이
2026년 9월 빅데이터 관련 자격증 빅데이터 처리 기술(Hadoop, Spark 등) 및 분석 기법 개요 학습, 자격증 유형에 맞는 문제 풀이

단순 합격을 목표로 하는 것이 아니라, 각 자격증에서 다루는 개념을 실제 프로젝트에 접목할 수 있도록 이해 중심의 학습을 지향한다. 특히 ADsP와 빅데이터 자격증 학습 과정에서 습득하는 데이터 분석 역량은 방학 중 개발할 프로젝트에도 적극 활용할 계획이다.

04

방학 중 프로젝트 계획

게임 데이터 관리 및 수익 분석 웹 시스템

방학 기간 중 HTML, CSS, JavaScript를 활용하여 데이터 관리 중심의 웹 애플리케이션을 자력 개발할 계획이다. 본 시스템은 온라인 게임 내 여러 캐릭터의 활동 기록과 수익 데이터를 체계적으로 저장·조회·분석하기 위한 목적으로 기획되었으며, 실제 본인이 직접 사용하는 것을 전제로 개발한다. 단순한 정적 웹페이지가 아닌, 데이터의 흐름과 상태 관리를 직접 설계하고 구현하는 실질적 소프트웨어 개발 경험을 목표로 한다.

  • 캐릭터별 정보(직업, 레벨 등) 등록 및 수정·삭제 기능
  • 일별·주별 활동 기록 입력 및 이력 조회
  • 수익 자동 계산 및 캐릭터별 누적 수익 집계
  • 기간별 수익 비교 통계 및 그래프 시각화 (Chart.js 활용 예정)
  • 전체 데이터 대시보드 구성 (요약 카드, 순위, 추이 등)
  • JSON 기반 데이터 내보내기·불러오기(백업·복원) 기능

이 프로젝트를 통해 프론트엔드 상태 관리 설계, 데이터 시각화 라이브러리 활용, 사용자 경험(UX)을 고려한 UI 구성 등 실무와 직결되는 역량을 집중적으로 강화할 계획이다.

포트폴리오 구축 계획

프로젝트 수행 과정에서 설계 내용, 구현 과정, 발생한 문제와 해결 과정을 꾸준히 기록하여 취업 활동에 실질적으로 활용할 수 있는 포트폴리오를 구축할 계획이다. 완성된 결과물만을 보여주는 데 그치지 않고, 어떤 고민을 거쳐 설계를 결정했는지, 어떤 오류를 만나 어떻게 해결했는지를 함께 담아 성장 과정 자체가 드러나는 포트폴리오를 지향한다.

  • 프로젝트 개요, 기획 배경 및 개발 목적
  • 주요 기능 소개 및 기술 스택 정리
  • 데이터베이스 설계 내용 (ERD, 테이블 정의서 포함)
  • 핵심 기능 구현 과정 및 트러블슈팅 사례
  • 결과 화면 캡처 및 시연 영상 자료
  • GitHub 저장소 연계를 통한 커밋 이력·코드 공개

GitHub를 중심으로 코드와 문서를 함께 관리하고, 각 프로젝트별 README를 충실하게 작성하여 코드를 보지 않더라도 프로젝트의 성격과 기여도가 명확히 전달될 수 있도록 준비할 예정이다.
